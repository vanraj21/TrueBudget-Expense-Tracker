# 💰 TrueBudget - Expense Tracker

A modern and intuitive personal finance management application built with Flutter that helps users manage income, expenses, and financial insights through a clean and responsive user interface.

## ✨ Features

- Secure User Authentication
- Beautiful Dark UI
- Dashboard Overview
- Income & Expense Management
- Add/Edit/Delete Transactions
- Transaction History
- Financial Analytics
- Category-wise Expense Breakdown
- Multi-Currency Support
- User Profile Management
- Responsive Material Design
- Bottom Navigation
- Premium Card UI
